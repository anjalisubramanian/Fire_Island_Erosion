# Fire_Island_Erosion

Within this repository, four files can be found:
1. The "Percent_erosion_deposition" file produces a stacked bar graph showing the percent of pixels across Fire Island experiencing erosion or deposition annually.
2. The "DEM_histogram" file produces a grouped histogram showing the distribution of elevation across Fire Island over a range of years.
3. The "Slope_KDE" file produces overlapping kernel density estimates showing the distribution of slope across Fire Island over a range of years.
4. The "Aspect_correlation_elevation" file produces a scatterplot showing the correlation between elevation change and aspect on Fire Island in 2013.
